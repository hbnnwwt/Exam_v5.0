# Design Context

## Users

- **Primary users**: 面试场内的秘书 (exam room clerks)
- **Context**: 控制考试流程，在面试现场操作考试系统
- **Job to be done**: 高效、可靠地控制面试流程，确保考试顺利进行

## Brand Personality

- **Three words**: 专业 · 简洁 · 可靠
- **Tone**: 务实、高效、低调
- **Emotional goal**: 让操作者专注于流程控制，不因界面设计而分心

## Aesthetic Direction

- **Visual direction**: 简洁专业 — 功能优先，避免花哨
- **Theme**: 优先浅色模式，深色模式作为备选
- **Anti-references**: 避免 AI 生成风格的配色（渐变色、玻璃拟态、彩虹色）、避免过度动画和装饰性元素
- **Reference**: 传统办公软件风格（如 Windows 系统自带工具）

## Design Principles

1. **功能优先** — 界面服务于操作效率，装饰性元素最小化
2. **清晰可辨** — 重要信息一目了然，状态变化明确
3. **操作可信** — 按钮操作有明确反馈，流程状态可预期
4. **朴素整洁** — 无渐变、无阴影滥用、无多余动画
5. **专业稳重的视觉** — 避免活泼轻浮的设计元素
