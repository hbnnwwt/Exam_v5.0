数据库连接问题

Q: 系统显示"数据库连接失败"怎么办？
  1. 确认服务器正常运行（检查控制台是否有错误信息）
  2. 刷新页面重新连接
  3. 检查数据库文件是否存在于 assets/data 目录
  4. 检查数据库文件权限是否正确
  5. 如果问题仍然存在，请联系技术支持

题目抽取问题

Q: 抽取题目时显示"无可用题目"？
  1. 进入题库管理，检查是否有足够的题目
  2. 确认题目没有全部被标记为"已使用"
  3. 可以通过系统重置清除使用状态
  4. 如果题库为空，需要先添加题目
  5. 检查题库文件是否损坏，必要时恢复备份

计时器问题

Q: 计时器不工作或显示异常？
  1. 点击"重置计时"按钮
  2. 检查系统设置中的时间配置
  3. 刷新页面重新初始化
  4. 确认浏览器没有禁用 JavaScript
  5. 尝试使用其他浏览器

状态恢复问题

Q: 如何恢复到某个考生的考试状态？

系统会自动保存每位考生的进度。刷新页面后，系统会自动恢复到最后一位考生的状态，包括当前步骤和已抽取的题目。无需手动操作，系统会在页面加载时自动检测并恢复上一次考试状态。

AI 出题问题

Q: AI 出题失败或连接超时？
  1. 检查 AI 服务是否正常运行
  2. 确认网络连接稳定
  3. 检查 API 配置是否正确
  4. 尝试重新启动 AI 服务
  5. 如果超时时间过短，可以在设置中增加超时时间
  6. 查看控制台错误日志以获取更多信息

数据备份问题

Q: 如何备份题库数据？

题库数据存储在 assets/data 目录下的 SQLite 数据库文件中（通常为 questions.db 或类似文件名）。备份方法：直接复制该数据库文件到安全位置即可。建议定期备份，并在进行系统升级前备份所有数据。
