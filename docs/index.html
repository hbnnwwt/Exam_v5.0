快捷键参考

Keyboard Shortcuts

考试系统支持键盘快捷键，可快速完成开始考试、切换步骤、抽取题目、计时控制等操作。

主要快捷键
  • Space 开始考试
  • ← → 上一步 / 下一步
  • D 抽取题目
  • T P R 计时控制
  • Ctrl+Enter 完成考试
