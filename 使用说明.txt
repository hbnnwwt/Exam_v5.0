================================================================================
                    研究生复试系统 - 使用说明
================================================================================

【系统简介】
本系统是基于 Vue 3 + Flask 架构的研究生复试面试管理系统。
支持面试流程控制、计时器、题库管理、随机抽题、考试导出等功能。

================================================================================
                        首次使用（三步走）
================================================================================

第一步：安装后端依赖
--------------------
双击运行：setup.bat

该脚本会自动完成：
  - 检测并配置 Python 环境（支持便携版/虚拟环境/系统Python）
  - 安装所需依赖包
  - 创建必要目录（data、images、logos、uploads、logs）
  - 初始化数据库

完成后会提示 "Setup completed successfully!"


第二步：构建前端
--------------------
双击运行：build.bat

该脚本会自动完成：
  - 检测 Node.js，未安装则自动安装
  - 安装前端依赖（npm install）
  - 构建前端静态文件

完成后会提示 "Build completed!"


第三步：启动系统
--------------------
双击运行：run.bat

系统启动后会自动打开浏览器访问：http://localhost:5000


================================================================================
                        日常使用
================================================================================

启动系统
--------------------
双击运行：run.bat

系统将在 http://localhost:5000 运行，浏览器会自动打开。


停止系统
--------------------
在命令行窗口按 Ctrl+C，或直接关闭窗口。


================================================================================
                        开发者模式（可选）
================================================================================

如需实时调试前后端代码，运行：dev.bat

该模式会同时启动：
  - 后端服务：http://localhost:5000
  - 前端开发服务器：http://localhost:3000（支持热更新）

注意：开发模式需要已安装 Node.js 环境。


================================================================================
                        常见问题
================================================================================

Q: 运行 setup.bat 提示 Python not found？
A: 脚本会自动下载便携版 Python，请确保网络连接正常。

Q: 运行 build.bat 提示 Node.js not found？
A: 脚本会尝试通过 winget 自动安装 Node.js，或手动从 https://nodejs.org/ 下载。

Q: 运行 run.bat 后浏览器无法访问？
A: 请等待几秒让服务器完全启动，或手动访问 http://localhost:5000

Q: 如何修改端口号？
A: 编辑 backend/config.py 文件中的 HOST 和 PORT 配置。
