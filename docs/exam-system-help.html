🎯 系统概述

研究生复试流程控制系统是一个专为研究生复试面试设计的综合管理平台，提供完整的面试流程控制、题库管理、计时功能和考生记录管理。

核心功能：

  • 六步考试流程：中文自我介绍 → 英文自我介绍 → 英文翻译 → 专业问题 → 综合问答 → 考试结束
  • 智能题库管理：支持翻译题和专业题的分类管理，包含文本和图片内容
  • 随机抽题功能：自动随机选择题目，避免重复，确保公平性
  • 精确计时控制：每个环节独立计时，支持暂停、重置和自定义时长
  • 考生记录管理：自动记录每位考生的考试进度和抽取题目
  • 系统状态监控：实时显示数据库连接、当前步骤、计时器状态等

🖥️ 界面介绍

界面布局说明：

  1. 顶部标题栏：显示系统标题、学校Logo和当前学生信息
  2. 左侧流程面板（20%宽度）：显示六步考试流程和当前进度

  5. 底部状态栏：显示计时器和版权信息

考试流程面板：

步骤 名称 默认时长 说明
1 中文自我介绍 1分钟 考生用中文进行自我介绍
2 英文自我介绍 1分钟 考生用英文进行自我介绍

6 考试结束 - 总结评分，准备下一位考生

🔄 考试流程

开始新考试：

  1. 点击"开始考试"按钮或按Space键
  2. 系统自动生成学生编号并创建考试记录
  3. 进入第一步：中文自我介绍
  4. 左侧面板显示当前步骤，右侧控制按钮激活

步骤切换：

自动流程：使用"下一步"按钮按顺序进行，系统会自动保存进度和状态。
手动跳转：可以使用"上一步"返回之前的环节，或"完成考试"直接结束。

题目抽取（步骤3和4）：

  1. 英文翻译（步骤3）：点击"抽取题目"自动随机选择翻译题
  2. 专业问题（步骤4）：首先选择科目，然后随机抽取该科目的专业题
  3. 抽取的题目会显示在中央内容区，包含文本和图片内容
  4. 已抽取的题目会被标记为"已使用"，避免重复

⚙️ 系统设置

图2：系统设置界面

时间设置：

可以自定义每个考试环节的时长（以秒为单位），设置后立即生效。

🔧 常见问题

Q: 系统显示"数据库连接失败"怎么办？

解决方案：
  1. 确认服务器正常运行（检查控制台是否有错误信息）
  2. 刷新页面重新连接
  3. 检查数据库文件是否存在于assets/data目录

Q: 抽取题目时显示"无可用题目"？

解决方案：
  1. 进入题库管理，检查是否有足够的题目
  2. 确认题目没有全部被标记为"已使用"
  3. 可以通过系统重置清除使用状态

Q: 计时器不工作或显示异常？

解决方案：
  1. 点击"重置计时"按钮
  2. 检查系统设置中的时间配置
  3. 刷新页面重新初始化

Q: 如何恢复到某个考生的考试状态？

解决方案： 系统会自动保存每位考生的进度。刷新页面后，系统会自动恢复到最后一位考生的状态，包括当前步骤和已抽取的题目。
