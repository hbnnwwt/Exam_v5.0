考试数据导出指南

研究生复试流程控制系统 · Exam Export Guide

统计概览

图 1：考试导出主界面

统计数据面板

  • 总学生数：参与考试的学生总数统计
  • 已完成：成功完成考试的学生数量
  • 完成率：以百分比形式展示的考试完成进度
  • 专业分布：各专业科目的考生选择人数分布
用途说明：统计数据面板帮助管理员快速了解当前考试的整体情况，便于进行考试进度监控和资源配置。

导出 Excel

导出操作步骤

  1. 在考试导出页面，点击右上角的"导出 Excel"按钮
  2. 系统自动生成包含所有考试记录的 Excel 文件
  3. 浏览器弹出下载对话框，保存文件到本地
  4. 使用 Excel 或 WPS 打开文件查看完整数据
Excel 内容：导出的 Excel 文件包含学生编号、考试状态、开始时间、结束时间、选择专业、抽取题目等完整信息。

生成 HTML 报告

HTML 报告生成步骤

  1. 在考试导出页面，点击"导出 HTML"按钮
  2. 系统生成包含完整考试记录的 HTML 文件包
  3. 下载完成后解压文件（包含 HTML 主文件和相关资源文件夹）
  4. 在浏览器中打开 HTML 文件即可查看精美报告
报告特点：HTML 报告采用响应式设计，支持图片展示，可在任何现代浏览器中良好显示。

导出格式说明

格式差异：考试导出系统支持 Excel 和 HTML 两种格式，各有不同的适用场景和特点。

Excel 格式特点

  • 数据完整：包含所有考试记录的完整字段信息
  • 便于分析：适合使用 Excel 进行数据筛选、排序和统计
  • 易于编辑：可自由修改单元格内容
  • 文件小巧：文件体积小，便于传输和存储

HTML 格式特点

  • 图文并茂：完整展示题目内容和图片
  • 样式精美：采用现代化页面设计，展示效果佳
  • 无需工具：只需浏览器即可查看
  • 适合存档：完整的单文件报告包，便于归档保存
使用场景推荐格式理由
数据分析Excel支持筛选、排序、公式等数据分析功能
报告展示HTML视觉效果好，支持图片和样式
数据存档Excel文件小，兼容性好，长期保存可靠
在线分享HTML可通过浏览器直接查看，无需安装软件
