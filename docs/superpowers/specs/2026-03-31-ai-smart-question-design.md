# 题库管理模块智能出题功能 - 设计文档

**项目**: 研究生复试面试系统
**日期**: 2026-03-31
**范围**: 题库管理模块 - AI 配置和智能出题

---

## 1. 设计目标

- **核心目标**: 接入 AI 模型辅助题库管理，提升出题效率
- **用户价值**: 面试官可快速生成高质量翻译题和专业题
- **成功标准**:
  - 支持多种 AI Provider (OpenAI/Claude/Gemini/自定义)
  - 单题智能生成 + 批量智能生成
  - 预览后导入工作流
  - 参考 lobehub 的配置 UI 范式

---

### 3.2 数据结构

**AI Provider 配置**:

```typescript
interface AiProvider {
  id: string;           // openai / claude / gemini / custom
  name: string;
  enabled: boolean;
  apiKey?: string;
  baseUrl?: string;
  models: string[];     // 可用模型列表
}
```

**AI 生成请求**:

```typescript
interface AiGenerateRequest {
  provider: string;
  model: string;
  type: 'translation' | 'professional';
  knowledge?: string;    // 知识点描述
  count: number;        // 生成数量
  context?: string;     // 参考文档内容
}
```

+------------------------------------------------------------------+

|                                                                  |
|  原文:                                                           |
|  +----------------------------------------------------------+   |
|  | The rapid advancement of artificial intelligence has     |   |
|  | brought unprecedented changes to various industries...       |   |
|  +----------------------------------------------------------+   |
|                                          [🤖 AI 生成]          |
|                                                                  |
|  参考译法:                                                     |
|  +----------------------------------------------------------+   |
|  | [AI 生成后显示于此]                                       |   |

**交互说明**:
1. 用户填写「原文」(或留空)
2. 点击「AI 生成」: 使用当前知识点/上下文智能生成
3. AI 返回多个候选，用户选择或编辑
4. 点击「AI 优化」: 基于反馈重新生成

## 5. API 设计

### 5.1 后端接口

| 端点 | 方法 | 说明 |
|------|------|------|
| `/api/ai/config` | GET/POST/PUT/DELETE | AI Provider 配置 CRUD |
| `/api/ai/test` | POST | 测试连接 |
| `/api/ai/generate` | POST | 单题 AI 生成 |
| `/api/ai/batch-generate` | POST | 批量 AI 生成 |

### 5.2 请求示例

**单题生成请求**:

```json
POST /api/ai/generate
{
  "provider": "openai",
  "model": "gpt-4o",
  "type": "translation",
  "context": "计算机专业英语",
  "source_text": "The rapid advancement of artificial intelligence"
}
```

**响应**:

```json
{
  "candidates": [
    "人工智能的快速发展给各行各业带来了前所未有的变革。",
    "人工智能的飞速发展已经为各行业带来了空前的改变。"
  ]
}
```

**批量生成请求**:

```json
POST /api/ai/batch-generate
{
  "provider": "openai",
  "model": "gpt-4o",
  "type": "professional",
  "knowledge": ["计算机网络", "操作系统"],
  "count": 20,
  "questions_per_set": 5
}
```
