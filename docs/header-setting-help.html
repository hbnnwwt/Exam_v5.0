🎯 功能概述

顶部设置是考试系统的外观定制功能，允许管理员自定义系统的顶部显示内容，包括考试标题和机构Logo，打造专属的考试界面。

核心功能：

  • 考试标题定制：设置显示在系统顶部的考试名称
  • 学校Logo上传：上传并显示学校标识
  • 学院Logo上传：上传并显示学院标识
  • 实时预览：即时查看设置效果
  • 格式支持：支持PNG、JPG、JPEG格式图片
  • 自动保存：设置后自动保存到数据库

  2. 考试名称设置：输入框用于设置系统标题

📝 考试标题设置

标题设置功能：

考试标题将显示在系统顶部的中央位置，是考生和考官看到的主要标识信息。

设置步骤：

  1. 在"考试系统标题"输入框中输入标题文字
  2. 标题会在下方的效果预览区实时显示
  3. 建议标题长度控制在30字符以内，确保显示效果
  4. 点击"保存设置"按钮保存更改
默认标题："北石化研究生复试流程控制系统"
建议格式：[学校名称] + [考试类型] + [系统名称]

🖼️ Logo上传管理

支持的Logo类型：

Logo类型 显示位置 建议尺寸 用途说明
学校Logo 顶部左侧 200x100px 显示学校标识
学院Logo 顶部右侧 200x100px 显示学院标识

上传步骤：

  1. 选择文件：点击对应的上传区域
  2. 文件选择：从本地选择图片文件
  3. 格式检查：确保文件为PNG、JPG或JPEG格式
  4. 预览确认：上传后在预览区查看效果
  5. 保存设置：点击保存按钮完成设置

文件要求：

  • 格式限制：仅支持PNG、JPG、JPEG格式
  • 文件大小：建议不超过2MB
  • 图片尺寸：建议200x100px，系统会自动调整
  • 图片质量：建议使用高清图片，确保显示效果

Logo管理操作：

  • 上传新Logo：点击上传区域选择新文件
  • 替换Logo：重新上传会自动替换原有Logo
  • 移除Logo：点击"移除"按钮删除当前Logo
  • 预览效果：所有更改会在预览区实时显示

👁️ 效果预览

预览功能说明：

效果预览区域会实时显示您的设置效果，包括：

  • 标题显示：考试标题的字体和位置
  • Logo布局：学校和学院Logo的位置和大小
  • 整体效果：顶部区域的完整视觉效果
  • 响应式预览：适配不同屏幕尺寸的显示效果
注意：预览效果与实际系统显示效果完全一致，可以作为最终确认的依据。

🎮 操作指南

主要操作按钮：

按钮 功能 使用场景
🔄 重置 恢复到默认设置 需要清除所有自定义设置时
💾 保存设置 保存当前所有设置 完成设置后确认保存

完整操作流程：

  1. 访问页面：从考试系统设置中进入顶部设置
  2. 设置标题：在输入框中修改考试标题
  3. 上传Logo：分别上传学校和学院Logo
  4. 预览效果：在预览区查看设置效果
  5. 保存设置：点击保存按钮确认更改
  6. 返回系统：设置完成后返回主系统

🔧 常见问题

Q: 上传Logo时提示格式不支持？

解决方案：
  1. 确认文件格式为PNG、JPG或JPEG
  2. 检查文件扩展名是否正确
  3. 尝试使用图片编辑软件重新保存文件

Q: Logo显示模糊或变形？

解决方案：
  1. 使用建议尺寸200x100px的图片
  2. 确保原图片清晰度足够
  3. 避免使用过小或过大的图片
  4. 保持Logo的宽高比例合适

Q: 设置保存后没有生效？

解决方案：
  1. 确认点击了"保存设置"按钮
  2. 刷新考试系统页面查看效果
  3. 检查网络连接是否正常
  4. 查看浏览器控制台是否有错误信息

Q: 如何恢复默认设置？

解决方案：
  1. 点击"重置"按钮清除所有自定义设置
  2. 系统会恢复到默认标题和无Logo状态
  3. 点击"保存设置"确认重置操作

💡 使用技巧

最佳实践建议：

  • Logo设计：使用简洁明了的Logo设计，避免过于复杂的图案

  • 备份文件：保留Logo原始文件，便于后续修改
