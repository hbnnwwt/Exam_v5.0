顶部设置使用指南

Header Settings Guide · URL 方式设置标题与 Logo

功能概述

图 1：顶部设置页面 — 学校 Logo、学院 Logo 与标题设置

顶部设置页面（路由 /header-settings）用于通过 URL 输入方式设置考试系统顶部栏的标题和 Logo 图片。与"考试设置"页面的文件上传方式不同，本页面通过粘贴图片链接来设置 Logo，操作更简洁。

  • 系统标题：自定义顶部栏显示的系统名称
  • 学院 Logo：通过 URL 设置学院标识图片
  • 学校 Logo：通过 URL 设置学校标识图片
  • 实时预览：输入后即时预览顶部栏效果
与"考试设置"页面的区别：本页面通过 URL 输入方式设置 Logo，适合已有图片链接的场景。而"考试设置"页面（/settings）通过文件上传方式设置 Logo，同时包含步骤属性编辑。两者修改的是同一组数据，任选其一使用即可。

URL 输入方式

设置系统标题

  1. 在"系统标题"输入框中输入标题文字
  2. 标题会显示在考试页面顶部栏中央
  3. 建议格式："XX 大学 XX 学院研究生复试面试"

设置 Logo（URL 方式）

  1. 将图片上传到图床或服务器，获取图片 URL 链接
  2. 在"学院 Logo URL"或"学校 Logo URL"输入框中粘贴链接
  3. 输入后，输入框下方会立即显示图片预览
  4. 如需更换，直接修改 URL 即可
  5. 如需删除，清空输入框内容

Logo 显示说明

Logo 字段显示位置说明
学院 Logo顶部栏左侧（第一个）学院标识，显示在标题左侧
学校 Logo顶部栏左侧（第二个）学校标识，显示在学院 Logo 右侧
图片来源：可以使用学校官网上的 Logo 链接、图床链接或本地服务器上的图片路径。确保 URL 可公开访问，否则图片可能无法加载。

实时预览

预览效果

页面底部有一个预览效果区域，实时显示顶部栏的最终效果：

  • Logo 排列：学院 Logo 和学校 Logo 按顺序显示在预览左侧
  • 标题显示：系统标题显示在预览右侧
  • 背景色：预览使用与考试页面相同的蓝色背景

保存设置

  1. 确认预览效果满意后，点击"保存设置"按钮
  2. 系统提示"保存成功"后，设置立即生效
  3. 返回考试页面（/）即可看到更新后的效果

图片要求

要求项目规格说明
图片格式PNG（推荐）、JPG、JPEG、GIF、WebP
建议尺寸宽度不超过 200px，高度不超过 80px
文件大小建议不超过 2MB
背景建议使用透明背景（PNG 格式）
比例横向比例最佳，适配顶部栏布局
注意事项：
  • 确保图片 URL 可公开访问，否则图片无法加载
  • Logo 设计应简洁明了，避免过于复杂的图案
  • 避免使用过大的图片，以免影响页面加载速度
  • 图片会通过 CSS object-fit: contain 自适应显示，不会变形
