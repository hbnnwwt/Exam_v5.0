AI 配置详细指南

AI Provider Configuration · 平台申请与系统配置完整教程

功能概述

本指南详细说明如何为研究生复试系统配置 AI 出题功能。与系统内帮助页面（/help）的简要介绍不同，本文档提供每个 AI 平台的完整申请流程和详细配置步骤。

  • 6 个内置提供商：OpenAI、Claude、Gemini、MiniMax、ModelScope、硅基流动
  • 自定义提供商：支持添加任何兼容 OpenAI/Anthropic API 格式的第三方服务
  • 智能出题：AI 可自动生成翻译题、专业问题，支持单题、套题、批量模式
  • 多提供商切换：同时配置多个提供商，按需切换或设置默认

图 1：AI 配置页面 — 提供商列表与配置面板

快速选择建议：
  • 追求质量：OpenAI (GPT-4o) 或 Claude (Sonnet 4)
  • 性价比优先：MiniMax 或 硅基流动（国产，价格低，中文能力强）
  • 国内可用：MiniMax、ModelScope、硅基流动（无需翻墙）

OpenAI API 申请

第 1 步：注册 OpenAI 账号

  1. 访问 platform.openai.com/signup
  2. 使用邮箱、Google 账号或 Microsoft 账号注册
  3. 完成邮箱验证

第 2 步：创建 API Key

  1. 登录 platform.openai.com/api-keys
  2. 点击 "Create new secret key"
  3. 输入 Key 名称（如 "Exam System"）
  4. 点击 "Create secret key"
  5. 立即复制并保存 Key（页面关闭后无法再次查看）

第 3 步：充值（如需要）

  1. 访问 Billing 页面
  2. 绑定信用卡（Visa / Mastercard）
  3. 新注册账号通常有 $5 免费额度（有效期 3 个月）
项目说明
官网platform.openai.com
默认模型gpt-4o
推荐模型gpt-4o（质量最佳）、gpt-4o-mini（性价比高）
Base URLhttps://api.openai.com/v1
参考价格gpt-4o: $2.50/百万输入 tokens，$10/百万输出 tokens
免费额度新账号 $5（3 个月内有效）
网络要求需要科学上网（中国大陆无法直接访问）
注意：OpenAI 不支持中国大陆手机号注册，需要使用邮箱注册。API 调用需要科学上网，或使用第三方代理（不推荐）。

Gemini (Google) API 申请

第 1 步：获取 Google AI Studio

  1. 访问 aistudio.google.com
  2. 使用 Google 账号登录

第 2 步：创建 API Key

  1. 点击左侧菜单 "Get API key"
  2. 选择或创建一个 Google Cloud 项目
  3. 点击 "Create API key"
  4. 复制并保存 Key

第 3 步：启用计费（如需要）

  1. 免费版有速率限制（15 RPM），日常使用通常足够
  2. 如需更高配额，前往 Google Cloud Console 启用计费

推荐模型gemini-2.0-flash（速度快）、gemini-1.5-pro（质量更好）
Base URLhttps://generativelanguage.googleapis.com/v1
参考价格免费版：15 RPM 限制；付费版：gemini-2.0-flash $0.10/百万输入 tokens
免费额度免费版有固定速率限制，无需付费即可使用
网络要求需要科学上网
优势：Gemini 免费版即可满足日常出题需求，速度快，适合批量生成题目。

系统内配置步骤

完成上述任一平台的 API Key 申请后，按以下步骤在系统中配置：

配置 AI 提供商

  1. 进入系统，导航到 "AI 配置" 页面（路由 /settings/ai）
  2. 点击要配置的提供商卡片（如 OpenAI、Claude 等）
  3. 在 API Key 输入框中粘贴刚才申请的 Key
  4. 展开 "高级选项" 可自定义 Base URL 和模型名称（通常使用默认值即可）
  5. 点击 "测试连接" 验证配置是否正确（仅内置提供商支持测试）
  6. 点击 "保存配置" 完成设置
提示：可以配置多个提供商，系统会根据需要切换。建议至少配置一个国内可用的提供商（如 MiniMax、硅基流动）作为备选。

自定义提供商

如果您的 AI 服务不在内置列表中，可以添加自定义提供商：

添加步骤

  1. 在 AI 配置页面点击 "添加自定义提供商" 卡片
  2. 填写 提供商名称（如 "公司内部 AI 服务"）
  3. 填写 API Key
  4. 填写 Base URL（如 https://your-api.example.com/v1）
  5. 填写 模型名称（如 your-model-name）
  6. 选择 API 格式：
    • OpenAI 兼容（/chat/completions）：适用于大多数兼容 OpenAI 格式的服务
    • Anthropic Messages（/v1/messages）：适用于兼容 Anthropic 格式的服务
  7. 点击 "添加" 完成添加
常见兼容服务：One-API、New-API、本地部署的 Ollama、vLLM 等均可通过自定义提供商接入。

设为默认提供商

设置方法：
  1. 在已配置的提供商配置面板中，勾选 "设为默认"
  2. 默认提供商在出题时会被自动选中，无需手动切换
  3. 可以随时更改默认提供商
建议：将性价比最高的提供商设为默认（如硅基流动或 MiniMax），需要高质量出题时手动切换到 OpenAI 或 Claude。

安全提示

API Key 安全须知：
  • 妥善保管：API Key 等同于您的账号密码，请勿泄露给他人
  • 不要提交到代码仓库：确保 API Key 不会通过 Git 等版本控制系统暴露
  • 定期更换：建议定期更换 API Key，减少密钥被盗用的风险
  • 限制 API 额度：为您的 AI 服务账号设置合理的调用额度限制，防止超额消费
  • 监控使用情况：定期查看各平台的用量统计，及时发现异常调用
隐私说明：您的 API Key 仅用于调用 AI 服务提供商的 API。系统不会将 Key 传输到任何第三方服务器。
