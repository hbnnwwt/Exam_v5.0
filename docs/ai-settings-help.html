AI 配置使用指南

AI Provider Configuration · 智能题库与生成

功能介绍

AI 配置功能为系统提供智能题库管理和题目生成能力，支持多种主流 AI 服务提供商，让您灵活配置最适合的 AI 服务。

  • 多提供商支持：同时配置多个 AI 服务商（OpenAI、Claude、Gemini、MiniMax、ModelScope、硅基流动等），随时切换
  • 智能题目生成：利用 AI 自动生成翻译题和专业问题，丰富题库内容
  • 自定义提供商：支持添加第三方兼容 API，可接入任何符合 OpenAI API 格式的服务
  • 默认提供商设置：设置默认 AI 提供商，生成题目时自动使用
  • 连接测试：配置完成后可立即测试连接是否正常，确保 API 可用
  • 安全存储：API Key 安全存储，仅用于调用 AI 服务，不会在本地保存

配置步骤

配置 AI 提供商

  1. 进入"AI 配置"页面
  2. 点击要配置的提供商卡片
  3. 填写 API Key（必填项）
  4. 展开"高级选项"可自定义 Base URL 和模型名称
  5. 点击"测试连接"验证配置是否正确
  6. 点击"保存配置"完成设置
提示：API Key 是您从 AI 服务商获取的凭证，请确保输入正确。测试连接按钮可以帮助您在保存前验证配置是否可用。

自定义提供商

添加自定义 AI 提供商

  1. 在 AI 配置页面点击"添加自定义提供商"
  2. 填写提供商名称（如 "我的 API"）
  3. 填写API Key（从第三方服务获取）
  4. 填写Base URL（第三方 API 的接口地址）
  5. 填写模型名称（要使用的模型标识）
  6. 点击"测试连接"验证
  7. 点击"保存"完成添加
兼容性说明：自定义提供商需要兼容 OpenAI API 格式。支持任何提供 OpenAI 兼容接口的第三方服务。

设为默认提供商

如何设置默认提供商：
  1. 在提供商卡片上点击"设为默认"按钮
  2. 或进入"高级选项"，勾选"设为默认"
  3. 默认提供商的卡片会显示星标标识
  4. 生成题目时，系统会自动使用默认提供商
注意：如果未设置默认提供商，系统会提示您选择要使用的 AI 提供商。

安全提示

API Key 安全须知：
  • 妥善保管：API Key 等同于您的账号密码，请勿泄露给他人
  • 不要提交到代码仓库：确保 API Key 不会通过 Git 等版本控制系统暴露
  • 定期更换：建议定期更换 API Key，减少密钥被盗用的风险
  • 使用环境变量：在生产环境中，建议使用环境变量而非硬编码方式存储 API Key
  • 限制 API 额度：为您的 AI 服务账号设置合理的调用额度限制
隐私说明：您的 API Key 仅用于调用 AI 服务提供商的 API，不会被保存到本地数据库或传输到其他服务器。
