Logo 设置说明

图 1：顶部设置主界面

支持的 Logo 类型

Logo 类型显示位置建议尺寸用途说明
学校 Logo顶部左侧200 x 100 px显示学校标识
学院 Logo顶部右侧200 x 100 px显示学院标识

上传步骤

  1. 选择文件：点击对应的上传区域
  2. 文件选择：从本地选择图片文件
  3. 格式检查：确保文件为 PNG、JPG 或 JPEG 格式
  4. 预览确认：上传后在预览区查看效果
  5. 保存设置：点击保存按钮完成设置

考试标题设置

说明：考试标题将显示在系统顶部的中央位置，是考生和考官看到的主要标识信息。

设置步骤

  1. 在"考试系统标题"输入框中输入标题文字
  2. 标题会在下方的效果预览区实时显示
  3. 建议标题长度控制在 30 字符以内，确保显示效果
  4. 点击"保存设置"按钮保存更改
默认标题："北石化研究生复试流程控制系统"
建议格式：[学校名称] + [考试类型] + [系统名称]

图片要求

格式支持：仅支持 PNG、JPG、JPEG 格式图片上传
要求项目规格说明
图片格式PNG、JPG、JPEG
建议尺寸200 x 100 px（系统会自动调整）
文件大小建议不超过 2MB
图片质量建议使用高清图片，确保显示效果
注意事项：
  • Logo 设计应简洁明了，避免过于复杂的图案
  • 确保 Logo 颜色与系统界面协调
  • 避免使用过小或过大的图片，以免变形或模糊
